Class DecoratorFilterMenuComponent

Component of decorator filter menu. Handles checking and unchecking visible decorators with appropriate actions. Filter menu is generated based on possible categories of decorators and decorator types for each category. If any more decorators are added or removed in the future this component should not need any additional changes as long as changes are made in CategoryEnum and DecoratorTypeEnum.
