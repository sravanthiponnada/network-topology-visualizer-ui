Class ForceGraphComponent

Main component of the graph topology application. On start it loads topology and decorators and store results in nodes and links attributes which are later used to construct and draw graph-visual of topology. Data can be also reloaded periodically or manually by user. Loaded data attribute is used for marking if getting data from server and parsing was already finished and its safe to construct graph-visual.
