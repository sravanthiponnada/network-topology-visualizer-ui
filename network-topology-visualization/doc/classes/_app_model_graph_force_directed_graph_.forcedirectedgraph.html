Class ForceDirectedGraph

Model of force directed graph-visual. Used for displaying graph-visual visualization and user interaction with it. Uses D3 and needs to get nodes, links and options (window size) on creation. D3 also does not support hierarchical graphs and it would be complicated to create new graph-visual for every subnetwork so this class implements multiple methods that fakes hierarchical structure and expanding/collapsing networks by removing and adding links and nodes and using forces in cooperation with structured model of nodes. This way, multi-layered hierarchies are supported.
